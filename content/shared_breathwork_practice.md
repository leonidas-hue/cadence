# Shared — Breathwork Practice Mode

**Type:** On-demand standalone (used by all variants + breathwork add-on)
**Voice direction:** Same voice as variant sessions. For pure breath-pacing tracks, the voice gives just minimal cues — no commentary.

This module produces three on-demand breath sessions the user can launch any time. They live in the "Practice" tab of the app, separate from the daily structured program.

---

# Pattern 3 — "Settle" (4-7-8 breathing)

**Duration:** ~5 min
**Use case:** Insomnia, acute stress, evening wind-down. Strongest parasympathetic effect of the three.
**Total breaths:** ~16
**Note:** This pattern is intense for new users. The 7-second hold can feel difficult. Tell beginners it's normal to feel slightly lightheaded the first few times.

---

Four breaths. Four-seven-eight breathing.

[PAUSE 2s]

[PAUSE 2s]

## Production notes

- Three separate audio files: `breath_calm.mp3`, `breath_ground.mp3`, `breath_settle.mp3`
- The "silent" segments in each can either be true silence or a very low-volume sound bed (the app can let users toggle this).
- For users with the optional camera-HRV biofeedback feature (later phase): the breath patterns sync with the HRV display so the user can see their HRV climb in real time during the practice.
- Pattern 3 (4-7-8) needs a "first time?" tooltip in the UI explaining it's the most demanding pattern.
