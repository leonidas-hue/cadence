# Pre-Session Prep — "Ready"

**Type:** Standalone audio (universal across all variants)
**Duration:** ~10 min (with optional 6-min "short" version markers)
**When to use:** 30–60 min before partnered sex
**Voice direction:** Same voice as variant sessions. Even slower than usual. No urgency. No motivational tone. Think "settled" rather than "psyched up." This is parasympathetic, not pre-game.

This script is the primary differentiator vs Relatio and MYHIXEL. Neither has a true in-session or pre-session companion. The goal here is to land users in a parasympathetic state with their attention in their body — without making them think about sex performance at all.

Content philosophy: we never explicitly mention "lasting longer" or "control" inside the script itself. The whole point is to take the user *off* the performance frame. The framing happens in the marketing. Inside the audio, we just work the body.

## Production notes

- Record at slightly lower volume than other sessions. Users will be listening on headphones in private.
- Add 5 seconds of silence at the very end before fade — gives users time to surface gracefully.
- Consider a low-volume ambient bed (very soft pink noise or distant ocean) under the entire track. Test both with-bed and dry versions in beta — some users find ambient helpful, others find it intrusive. Setting toggle if needed.
- File naming: `prep_long.mp3` (full ~10 min) and `prep_short.mp3` (cut version ~6 min)
- Both versions ship in v1; the user picks based on time available.
